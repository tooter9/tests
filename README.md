# ExifTool Manager

Удобный инструмент для работы с метаданными (EXIF/XMP/IPTC) через красивый терминальный интерфейс. Специально сделан для **iSH** (iOS терминал).

Использует системный **ExifTool** — не содержит собственной обработки файлов, только удобный интерфейс поверх него.

---

## Установка

### 1. Клонировать репозиторий

```sh
git clone https://github.com/ВАШ_НИК/exiftool-manager.git
cd exiftool-manager
```

### 2. Запустить установщик

```sh
sh install.sh
```

Установщик автоматически поставит:
- `python3`
- `pip3`
- `exiftool`
- Python-зависимости (`rich`)

### Ручная установка (если нужно)

```sh
# В iSH (Alpine Linux)
apk add python3 py3-pip exiftool

# Зависимости
pip3 install -r requirements.txt
```

---

## Запуск

```sh
python3 main.py
```

## Возможности

| Режим | Описание |
|-------|----------|
| Просмотр | Все теги файла, сгруппированные по категориям |
| Редактирование | Изменить один или сразу несколько тегов |
| Очистка | Удалить все или конкретные метаданные |
| GPS | Просмотр координат + ссылка Google Maps, запись координат, удаление |
| Копирование | Перенести теги из одного файла в другой |
| Пакетная обработка | Работа с целой папкой файлов |
| Экспорт | Сохранить метаданные в JSON или CSV |
| Переименование | Переименовать файл по дате из EXIF |
| Справка | Поиск информации по тегам |

---

## Поддерживаемые форматы

Все форматы которые поддерживает ExifTool:
JPEG, PNG, TIFF, HEIC, GIF, BMP, WebP, MP4, MOV, AVI, MKV, PDF, DOCX, MP3, FLAC и многие другие.

## Зависимости

- **Python** 3.8+
- **ExifTool** (системный, устанавливается через `apk add exiftool`)
- **rich** 13.0+ (красивый вывод в терминале)

## Безопасность

Инструмент **не содержит собственного кода обработки файлов** — все операции выполняются через системный `exiftool`. Это исключает возможные уязвимости от самописного парсера метаданных.

---

## Примечания для iSH

- iSH запускает Alpine Linux, поэтому используется `apk` для установки пакетов
- Убедись что iSH имеет доступ к папке с фотографиями через Files
- Пути к файлам указывай полные или с `~/`
